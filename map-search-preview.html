🧪 How to Test

Step 1: Start Development Server

Run the following command in your terminal:

npm run dev

Step 2: Open Application

Visit http://localhost:3000 in your web browser

Step 3: Test the Search Feature

1. Click on "Enter pickup location" or "Enter drop location"

2. Type a location name in the search box (e.g., "Mumbai", "Haji Ali", "Delhi Airport")

3. Wait for results to appear (should take ~400ms after you stop typing)

4. Click on any result to jump to that location

5. The address bar should update with the selected location

6. Click "Confirm Pickup/Drop Location" to save

What to Look For:

✅ Search box appears at the top of the map picker

✅ Results dropdown shows up smoothly with animation

✅ Clicking a result zooms the map to that location

✅ Address updates automatically

✅ Search works together with manual map dragging

✅ Confirm button works after location is selected

Try These Searches:

• Mumbai • Haji Ali, Mumbai • Delhi Airport • Bangalore Metro • India Gate • Taj Mahal • Your local landmark
